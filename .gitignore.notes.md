## ⚠️ Security Reminders

1. **Never commit:**
   - `.env` files with real passwords
   - SSH private keys (`.pem`, `.key` files)
   - Real credentials in config files

2. **Before committing `config/prod.yaml`:**
   - Change the default password from `"password"` to a placeholder
   - Or use environment variables for sensitive values

3. **Use GitHub Secrets for:**
   - Docker Hub credentials
   - EC2 SSH keys
   - Application passwords
   - Database passwords
